# BM Quest KSSM Style Guide

## Learning Language

BM Quest KSSM follows this principle:

Malay first. English as a learning aid.

Example:

PETIKAN
Reading Passage

KOSA KATA
Vocabulary

## UI Principles

- Mobile-first
- Large readable cards
- Clear primary action per screen
- Rounded, friendly interface
- Encouraging learning tone

## Component Rules

- Prefer reusable components.
- Avoid large components over 150 lines where possible.
- Keep pages focused on layout and data composition.
- Move business logic into services.
